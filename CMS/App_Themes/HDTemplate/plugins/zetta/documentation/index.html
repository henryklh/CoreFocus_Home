Classes

1. Main

  1. .zetta-menu - main menu class;
  2. .zm-active - active menu item;
  3. .zm-right-item - right aligned item. Accept to main menu items only;

2. Logo

  1. .zm-logo - main logo class;

3. Icons

  1. .zm-caret - dropdown caret;
  2. .zm-single-icon - single icon (without label) item;

4. Dropdown

  1. .zm-drop-left - left shift dropdown menu class;
  2. .zm-multi-column - multicolumn dropdown;
  3. .zm-right-align - right align dropdown;

5. Content Dropdown

  1. .zm-content - auto width content class;
  2. .zm-content-full - full content class;
  3. .zm-right-align - right align dropdown;

6. Grid System

  1. .zm-row;
  2. .zm-col;
  3. .[c-1...c-12] - columns classes from 1 to 12 cells;
  4. .zm-grid-wrapper - grid column wrapper;

7. Search Place

  1. .zm-search;

8. Other

  1. .zm-button - styled button class;
  2. .zm-video - wrapper for iframe (video, maps...);

jQuery Parameters

1. position (string)

Default: -

One of preset menu positions:

  1. left
  2. right
  3. bottom
  4. other value means that the menu on top

$(".zetta-menu").zettaMenu({
   position: "left"
});


2. showOn (string)

Default: click

One of preset show menu variations:

  1. hover
  2. click
  3. toggle

$(".zetta-menu").zettaMenu({
   showOn: "hover"
});


3. responsive (string)

Default: switch

One of preset menu responsive view:

  1. simple
  2. switch
  3. stack
  4. switch-margin
  5. stack-margin
  6. Use another value if you no want responsive

$(".zetta-menu").zettaMenu({
   responsive: "stack-margin"
});


4. fullWidth (boolean [true, false])

Default: true

Use auto or full width menu. 

$(".zetta-menu").zettaMenu({
   fullWidth: false
});


5. sticky (boolean [true, false])

Default: false

Menu sticks to one of sides (depending on the position value) when it reaches the edge of the window. 

$(".zetta-menu").zettaMenu({
   sticky: true
});


6. fixed (boolean [true, false])

Default: false

Menu is fixed to one of sides (depending on the position value). 

$(".zetta-menu").zettaMenu({
   fixed: true
});


7. effect (object)

Default: 

effect: {
   name: "fade",
   easing: "ease",
   speed: 500,
   delay: 0
}

If you do not want the effect, use this code 

$(".zetta-menu").zettaMenu({
   effect: false
});


7.1. name (string)

Default: fade

One of preset show effects:

  1. fade
  2. slide-up
  3. slide-down
  4. slide-left
  5. slide-right

$(".zetta-menu").zettaMenu({
   effect: {
      name: "slide-up"
   }
});
7.2. easing (string)

Default: ease

One of preset show effects:

  1. ease
  2. ease-in
  3. ease-out
  4. linear
  5. cubic-bezier(...) - you can generate it here

$(".zetta-menu").zettaMenu({
   effect: {
      easing: "cubic-bezier(0.230, 1.000, 0.320, 1.000)"
   }
});
7.3. speed, delay (int)

Default: 500, 0

Values in milliseconds. 

$(".zetta-menu").zettaMenu({
   effect: {
      speed: 100,
      delay: 300
   }
});

Main

First of all You should include one of the styles into your project, placed in folder themes. Next step - is include jQuery, zm.jquery.js and initialize plugin after menu. Also, if you plan to use the icons, you must connect the styles Font Awesome. The structure of your document will be like this:

<!DOCTYPE html>
<html>
<head>
   <!-- Zetta Menu -->
   <link rel="stylesheet" href="zetta-menu.min.css">
   <script src="jquery.js" type="text/javascript"></script>
   <script src="zm.jquery.js" type="text/javascript"></script>

   <!-- Font Awesome -->
   <link rel="stylesheet" href="font-awesome.min.css">

</head>

<body>
   <!-- Your Menu Here -->
   <ul class="zetta-menu">...</ul>

   <!-- ZM Plugin Initialization -->
   <script>
      $(".zetta-menu").zettaMenu();
   </script>
</body>
</html>

Grid System

Zetta Menu uses 12-column grid system. You can use grid-system in content dropdown only (.zm-content or .zm-content-full).

  3. .zm-content-full
  4. .zm-row
  5. .zm-col
  6. .c-4
  7. .c-8
